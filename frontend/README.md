# Smart Grocery Cart management system

A hybrid python-C shopping cart that uses linked lists, greedy algorithms and priority based budget optimisation for intelligent cart management.

## Features of this system

> Smart Grocery shopping cart with budget tracking 
> Priority based item replacement/removal
> Category-wise analysis of budgets
> implementation of Linked List
> Frontend - Python Streamlit
> Backend - C integration using cytypes

## Tech Stack

> C
> python
> Streamlit
> Ctypes
